  三、ETH / 更长周期是否是好方向？
                                                                                                                                                                     
  ETH（同 15m）：ETH 与 BTC 相关性 0.85+，adverse selection 几乎相同，且 Polymarket 上 ETH 流动性更低。边际改善有限，不值得单独切换。
                                                                                                                                                                     
  1h BTC：这是最值得测试的方向。                        
                                                                                                                                                                     
  15m (900s)  + cutoff=240s → 利用率 73%，BTC 短期方向性强（残仓发生率高）                                                                                           
  1h (3600s)  + cutoff=240s → 利用率 93%，1h 方向不确定性更高（流量更均衡）                                                                                          
                                                                                                                                                                     
  1h 的三个直接好处：                                                                                                                                                
  1. 利用率 +20%：配对时间更充裕                                                                                                                                     
  2. 1h 方向不可预测 → YES/NO 流量更均衡 → 残仓发生率更低（Real2 R2 这类单边轮次会减少）                                                                             
  3. 更宽的 YES+NO 价差 → 可以捕获更大利润空间                                          
                                                                                                                                                                     
  零代码改动，只需改 .env 的 POLYMARKET_MARKET_SLUG=btc-updown-1h。
